# Polices

Pour activer le logo et la typo display, ajouter les fichiers woff2 dans ce dossier :

- `CormorantGaramond-Light.woff2` (poids 300)
- `CormorantGaramond-Regular.woff2` (poids 400)

Sources : [Google Fonts](https://fonts.google.com/specimen/Cormorant+Garamond) ou outils de conversion woff2.
